**Setting Up Project**
<br/>
`git clone https://github.com/FgrReloaded/FullStack_Project.git`
<br/>
`cd FullStack_Project`

**Setting Up Backend**
<br/>
`cd backend`
<br/>
`python -m venv env`
<br/>
`source env/bin/activate`
<br/>
`pip install -r requirements.txt`

**Setting Up Backend**
<br/>
`cd frontend`
<br/>
`npm i`
